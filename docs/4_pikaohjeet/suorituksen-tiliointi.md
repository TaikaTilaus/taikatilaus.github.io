# Virheellisen suorituksen tiliöinti

### 1. Avaa Suoritukset-välilehti ja valitse Hae vain käsittelyä vaativat. Paina Hae

![1. Suoritukset välilehti](/img/pikaohjeet/virheellisensuorituksentiliointi1.png)

### 2. Alle tulee lista virheellisistä suorituksia. Info-kentässä lukee selvitys, miksi suoritus ei kohdentunut laskulle

![2. Virhehaun listaus ja infokentän merkitys](/img/pikaohjeet/virheellisensuorituksentiliointi2.png)

### 3. Valitse Laskun numero, mitä haluat käsitellä

![3. Laskun valinta](/img/pikaohjeet/virheellisensuorituksentiliointi3.png)

### 4. Laskunäkymästä klikkaa Liitä suoritus-painiketta

![4. Laskunäkymä](/img/pikaohjeet/virheellisensuorituksentiliointi4.png)

### 5. Suorituksen liittäminen. Ohjelma hakee virheellisen suoritus IDn. Kun ID on oikein, paina Hae

![5. Suorituksen liittäminen](/img/pikaohjeet/virheellisensuorituksentiliointi5.png)

### 6. Täytä suorituksen tiedot avautuvaan kenttään ja paina Tallenna

![6. Suorituksen käsittely](/img/pikaohjeet/virheellisensuorituksentiliointi6.png)

### 7. Laskulle tulee näkyviin tiliöinti ja lasku muuttuu maksetuksi

![7. Valmis tiliöinti](/img/pikaohjeet/virheellisensuorituksentiliointi7.png)
