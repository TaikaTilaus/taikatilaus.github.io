# Suorituksen liittäminen laskuun


### 1. Miksi suoritus liitetään manuaalisesti?

Suoritus liitetään manuaalisesti laskuun esim. kun asiakas on maksanut vain osan laskusta tai suorituksessa oli virheellinen tilinumero, jonka vuoksi sitä ei pystyttä automaattisesti yhdistämään laskuun.

### 2. Suoritukset-välilehdeltä näet viallisia suorituksia. Otetaan esimerkkinä käsittelyyn suoritus, jossa oli väärä viitenumero. Ota talteen SuoritusID.
