# Laskutuspyyntöjen muodostaminen laskuiksi ja laskujen lähetys manuaalisesti

### 1. Aloitetaan luomalla laskutuspyynnöistä laskut. Mennään Laskutus-välilehdelle. Haetaan laskutuspyynnöt. Jos etsit tietyn asiakkaan laskutuspyyntöjä, hae asiakasnumerolla.

![1. Laskutus-välilehti](/img/pikaohjeet/laskut-lah1.png)

### 2. Valitse hakutuloksista haluamasi laskutuspyyntö klikkaamalla kenttä aktiiviseksi (tai paina Valitse kaikki, jos haluat luoda laskuiksi kaikki haun antamat laskutuspyynnöt). Deaktivoi Esikatselu-kenttä. Suorita laskutusajo.

![2. Laskutus-välilehti - Hakutulos](/img/pikaohjeet/laskut-lah2.png)

### 3. Näytölle tulee todiste suorituksesta ja PDF-latauslinkki, josta voit tarkastella luotuja laskuja.

![3. Laskutus-välilehti - Laskujen muodostamisajo](/img/pikaohjeet/laskut-lah3.png)

### 4. Siirrytään Laskujen lähetys -välilehteen. Valitse laskun toimitustapa. Tässä esimerkissä se on SÄHKÖPOSTI. Luotu lasku näkyy ylimmäisenä. Valitse se. Paina Lähetä lasku.
![4. Laskujen lähetys -välilehti](/img/pikaohjeet/laskut-lah4.png)

### 5. Muokkaa laskun saateviestiä, jos tarvetta. Ota esikatselu pois päältä. Lähetä laskut.
![5. Laskujen lähetys -välilehti - Aineiston luonti](/img/pikaohjeet/laskut-lah5.png)

### 6. Näytölle tulee vahvistusviesti laskujen lähetyksestä.
![6. Laskujen lähetys -välilehti - Vahvistusviesti](/img/pikaohjeet/laskut-lah6.png)
